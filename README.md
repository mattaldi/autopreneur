## 🌟 Apa itu Autopreneur?

Autopreneur adalah aplikasi pintar yang:
- ✅ **Riset Pasar Otomatis** - AI menganalisis ide bisnis Anda dan memberi skor 0-100
- ✅ **Buat Produk Digital** - Generate PDF panduan + data siap pakai
- ✅ **15+ Jenis Produk** - Dari caption Instagram hingga kalkulator pajak
- ✅ **100% Bahasa Indonesia** - Dibuat khusus untuk pasar Indonesia
- ✅ **Mudah Digunakan** - Cukup pilih menu, tidak perlu coding!

## 💻 Cara Install (Sekali Saja)

### Yang Dibutuhkan:
- Python (gratis download di python.org)
- API Key OpenAI (daftar di openai.com)
- wkhtmltopdf (untuk buat PDF)

### Langkah Install:

#### 1. Download Autopreneur
```bash
# Buka Command Prompt/Terminal
cd C:\
mkdir code
cd code
git clone https://github.com/mattaldi/autopreneur.git
cd autopreneur
```

#### 2. Setup Python
```bash
# Buat environment
python -m venv venv

# Aktifkan (Windows)
venv\Scripts\activate

# Install library
pip install -r requirements.txt
```

#### 3. Masukkan API Key
Buat file `.env` di folder autopreneur:
```
OPENAI_API_KEY=sk-xxxxxxxxxxxxxxx
```

#### 4. Install wkhtmltopdf
Download dari: https://wkhtmltopdf.org/downloads.html
Install seperti aplikasi biasa.

**Selesai! Siap digunakan!** 🎉

## 🆘 Bantuan

### Kalau Error:

**"API Key tidak ditemukan"**
- Cek file `.env` sudah ada
- Pastikan format: `OPENAI_API_KEY=sk-xxx`

**"Rate limit exceeded"**
- Tunggu beberapa menit
- Atau upgrade plan OpenAI

**"PDF gagal dibuat"**
- Install wkhtmltopdf
- Restart Command Prompt
